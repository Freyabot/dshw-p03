7 结论

基于本次作业完整流程与 output/ 结果文件，可得到以下结论：

  1. 基准模型 M1 显示 NPR 对 Lev 显著负向，核心证据支持优序融资理论。
  2. 加入宏观变量后的 M1’ 与 M1 结果几乎一致，说明基准结论稳健。
  3. 产权异质性显著：SOE 的负向斜率绝对值高于非 SOE。
  4. 时变系数模型 M4 表明该关系强度随时间变化，但 2011 年后多数年份保持显著负向。
  5. M6 显示规模门槛异质性；M5 的平滑非线性证据相对较弱。

7.1 对作业四个核心问题的回答

  1. 理论检验：M1-M3 的综合证据整体更支持优序融资理论，且结论受产权性质影响。
  2. 时序稳定性：M4 显示斜率存在阶段变化，但方向基本稳定为负。
  3. 信息不对称机制：M5-M6 显示规模存在异质性，门槛结果提供了更清晰证据。
  4. IFE vs TWFE：加入 m2_growth 后 NPR 系数几乎不变，说明主结论不依赖该扩展设定。

7.2 局限性

  • M5 多项式交互不显著，对平滑非线性解释需谨慎。
  • M6 当前为门槛式网格稳健性实现，尚非完整 Hansen 自助法推断。
  • 现有结果仍以相关性证据为主，潜在内生性问题仍需进一步处理。

7.3 可继续优化的方向

  • 引入动态面板框架（如滞后杠杆、GMM）提升识别强度。
  • 增加融资约束代理变量，细化机制解释。
  • 对门槛效应做完整 bootstrap 置信区间推断。
  • 增加子样本（如 2015 年后）与行业分组稳健性检验。

总体而言，本研究在 2010-2025 年中国 A 股样本上得到一致证据：盈利能力上升对应杠杆率下降，且该关系在产权、时间与规模维度上表现出系统性异质性。
